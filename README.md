oauth2 package contains a client implementation for OAuth 2.0 spec.

## Installation

~~~~
go get golang.org/x/oauth2
~~~~

## Run

~~~
go run main.go
~~~

~~~
listening on port xxx
~~~

See pkg.go.dev for further documentation and examples.
